# 日本国 会社法（417AC0000000086）履歴リポジトリ

このリポジトリは、日本の「会社法」（法令ID: 417AC0000000086）の法改正履歴をGitのコミット履歴として管理する試みです。

## 概要

- **1コミット = 1法改正**: 1つの法改正に対して1つのGitコミットを対応させています。
- **タグ付け**: 各コミットはe-Govの「法令履歴ID（`law_revision_id`）」でタグ付けされています。
- **自動更新**: GitHub Actionsを利用し、定期的にe-Gov法令API（Version 2）から最新の改正情報を取得して自動でコミット・タグ付けを行います。

## ファイル構成

リポジトリには以下の法令データファイルが含まれます。

- `law.json`: e-Gov法令APIから取得した生データ（JSON形式）。
- `law.md`: JSONataを用いて、`law.json`からMarkdown形式に変換したデータ。
- `sync.mjs`: 自動同期および変換を行うzxスクリプト。
- `convert.jsonata`: JSONからMarkdownへの変換ルールを定義したJSONataファイル。

## データソース

法令データは、デジタル庁が提供する[e-Gov法令API（Version 2）](https://laws.e-gov.go.jp/api/2/redoc/)を利用して取得しています。
